#Grassland-Biodiversity-Data-Analysis
A Python-based data engineering and analytics project exploring the relationship between climate data and grassland yield across European and Canadian experimental sites.

An end-to-end data manipulation project using Python and Pandas to analyze environmental factors affecting agricultural yield.

#Project Overview
This project processes large-scale biodiversity experimental data from multiple sites across Europe and Canada. The goal was to clean, filter, and merge disparate datasets (Biomass and Climate) to visualize the impact of temperature on annual crop yield.

#Tech Stack
- Language: Python
- Libraries: Pandas (Data Wrangling), Matplotlib/Seaborn (Visualization), NumPy
- Environment: Jupyter Notebook

#Repository Structure
- `data/`: Contains raw CSV datasets and metadata.
- `notebooks/`: Contains the documented Python analysis (`.ipynb`).
- `Final_Plot.html`: A standalone export of the generated data visualizations.

#Key Takeaways
- Handled multi-site data with varying experimental start dates.
- Performed complex data cleaning to ensure numeric integrity across 50,000+ records.
- Created presentation-quality visualizations for environmental reporting.
